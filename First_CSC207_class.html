WEB

A website can be written in using three languages:

  • HTML :For making up the website content.
  • CSS :For styling the website.
  • JavaScript :For making the website interactive.

What is the Web?

The Web is basically a system of interenet servers that supports specially formarted documents. The documents are formmated in a markup language called HTML(HyperText Markup Language) thats supports links to other documents as well as graphics, audios and video files.

Internet: This is a masssive network of networks. It connects millions of computers together globally forming a network in which any computer can communicate with another computer as long as they are both connected to the internet.

Browser: This is a computer software that allows a client to interact and view the contents of a webpage. Examples are Opera, Firefox, Chrome etc.

Clients: These are devices which from a browser view or interact with a website.

Servers: This is a machine which contains all the information of a website.

How does a Webpage work?

A client uses a browser to view a webpage; the client must be connected to the internet and have a web browser. The client then type in the URL(Universal Resource Locator) in the address bar of the browser. The URL is now sent to the DNS(Domain Name System) server which searches for the IP(Internet Protocol) of the website's server. After locating the server the server sends back its information back to the client through the browser.

HTML

HTML means HyperText Markup Language. It is a markup language used to program websites.

Tags :These are set of characters which contain a formmated command for a webpage. A tag is used to define the structure on a HTML page. Tags are used in writing commands in HTML and are very neccesary. They are started with a less-than sign ( <) and closed with greater-than sign (>).

There two types of tags namely:

  1. Container tags
  2. Self Closing Tags
  1. Container Tags: This type of tags start with a open tag and ends with a closing tag.

    Syntax
    For example, The bold tag: It starts with < b> text... and ends with < /b>

  2. Self closing Tags: As the name implies it means it does not have a closing tag.

    Syntax
    It is written at the front of a line. < tag > text... It does not end with a closing tag.

Comment tag: A comment tag is basically used for documentation. It is written in the code to make a particular line more explainary for a person reading the code.

Syntax
< ! --text...-- >

Attributes: These are used to change the default appearance of a tag. To add an attribute to a tag; it must be included in the open tag.

Syntax
For example< tag name = value>
